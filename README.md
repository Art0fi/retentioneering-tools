# retentioneering tools

## Подготовка к работе

### Pre-commit

В проекте используется pre-commit хуки. Уставка хуков на локальную машину:

Pip
```commandline
pip install pre-commit
pre-commit install
```

Conda
```commandline
conda install -c conda-forge pre-commit
pre-commit install
```
